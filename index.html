Desafio DIO HTML

O HTML é a espinha dorsal de qualquer página na web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que diz ao navegador como estruturar o conteúdo. Sem o HTML, a internet como conhecemos não existiria.

Este documento demonstra o uso de diversas tags, desde as fundamentais até as obsoletas versáteis que compõem a web moderna.

Podemos utilizar a tag font (embora seja desencorajada no HTML5 moderno em favor do CSS) para alterar estilos básicos diretamente.

Na ciência, usamos muito subscrito (como em H2O) e sobrescrito (como em E=mc2).
