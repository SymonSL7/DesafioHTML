Desafio DIO HTML

O HTML (HyperText Markup Language) é a espinha dorsal de qualquer página na web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que diz ao navegador como estruturar o conteúdo. Sem o HTML, a internet como conhecemos não existiria, pois ele é o responsável por dar significado aos elementos que vemos na tela.

Através dele, podemos definir o que é um título importante, onde começa um parágrafo e como inserir imagens que tornam a experiência do usuário mais rica. Este guia prático foi criado para documentar o aprendizado das tags fundamentais.

Tags de Texto

< Voltar

H1 a H6: Estas tags são fundamentais para a hierarquia da informação. O h1 é o título principal da página, aquele que resume todo o assunto tratado. Os níveis subsequentes (h2, h3, até h6) servem para organizar subtópicos dentro do conteúdo. É uma boa prática de SEO (otimização para motores de busca) ter apenas um h1 por página.

P: A tag de parágrafo é a unidade básica de texto. Sempre que você desejar iniciar um novo bloco de ideias ou uma nova linha de raciocínio, deve envolver o texto na tag p. Isso garante que o navegador aplique o espaçamento correto entre os blocos.

Strong: Quando você deseja destacar uma palavra ou frase, a tag strong é a escolha ideal. Ela deixa o texto em negrito, mas seu propósito principal é semântico: ela avisa ao navegador e aos leitores de tela que aquele conteúdo tem uma importância maior ou urgência.

I: A tag i é utilizada para deixar o texto em itálico. Tradicionalmente, ela era usada apenas para fins visuais, mas hoje em dia é comum utilizá-la para termos estrangeiros, pensamentos ou termos técnicos que precisam de uma diferenciação visual sutil no corpo do texto.











Lista e Links

< Voltar

A (Âncora): Esta é a tag que conecta a web. Ela cria hiperlinks que podem levar o usuário para outros sites, para arquivos ou para pontos específicos da própria página, como fizemos no índice acima. O atributo "href" é obrigatório, pois indica o destino final do clique.

UL: A lista não ordenada é usada quando a ordem dos itens não altera o sentido da informação. Por padrão, o navegador exibe esses itens com pequenos círculos (bullet points). É muito utilizada para criar menus de navegação e rodapés de sites.

OL: Já a lista ordenada é numerada automaticamente pelo navegador (1, 2, 3...). Ela é perfeita para descrever processos, passos de uma receita ou qualquer conteúdo onde a sequência lógica seja obrigatória para o entendimento.

LI: A tag li (List Item) é a peça fundamental das listas. Ela nunca deve estar solta no código; deve sempre estar contida dentro de uma tag ul ou ol para que a estrutura seja válida e compreensível pelo navegador.











Mídia

< Voltar

IMG: A tag de imagem permite que você dê vida visual ao seu projeto. Ela é uma das poucas tags que não possui fechamento (não existe um /img). Para que ela funcione, você precisa de dois atributos principais:

  • SRC: Sigla para "source" (origem), onde você coloca o link ou o caminho da pasta onde a imagem está salva.
  • ALT: Sigla para "alternative" (texto alternativo), que descreve o que está na imagem para pessoas com deficiência visual e ajuda o Google a entender o conteúdo da foto.
